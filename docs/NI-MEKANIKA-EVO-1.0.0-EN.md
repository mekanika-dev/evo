### 1.3 Intended use {#13-intended-use}

Please read this user manual and all accompanying documents in their entirety before commissioning the machine.

If in doubt or if you need additional information, please do not hesitate to contact us before using the machine.

#### 1.3.1 Normal Use {#131-normal-use}

The MEKANIKA EVO CNC milling machine is a 3-axis CNC machine designed for:

<ul style="list-style-type: none; margin-left: 0.75em">
  <li><i class="fas fa-square-check" style="color: #4EBD7D;"></i> Milling wood and wood-based materials (plywood, MDF, OSB)</li>
  <li><i class="fas fa-square-check" style="color: #4EBD7D;"></i> Milling plastics (PVC, acrylic, polycarbonate)</li>
  <li><i class="fas fa-square-check" style="color: #4EBD7D;"></i> Milling non-ferrous metals (aluminum, brass)</li>
  <li><i class="fas fa-square-check" style="color: #4EBD7D;"></i> Milling composite materials (with special precautions)</li>
</ul>

#### 1.3.2 Prohibited Uses {#132-prohibited-uses}

The following uses are **PROHIBITED**:

<ul style="list-style-type: none; margin-left: 0.75em">
  <li><i class="fas fa-xmark" style="color: #d32f2f;"></i> Milling unknown composite materials or materials containing asbestos</li>
  <li><i class="fas fa-xmark" style="color: #d32f2f;"></i> Use in explosive atmospheres</li>
  <li><i class="fas fa-xmark" style="color: #d32f2f;"></i> Outdoor use or use in humid environments</li>
  <li><i class="fas fa-xmark" style="color: #d32f2f;"></i> Milling improperly secured workpieces</li>
  <li><i class="fas fa-xmark" style="color: #d32f2f;"></i> Milling beyond the machine's technical capabilities</li>
</ul>

### 2.3 Safety devices {#23-safety-devices}

#### 2.3.1 Emergency stop {#231-emergency-stop}

**Characteristics:**

- Type: Lockable push button
- Color: Red on yellow background
- Standard: EN ISO 13850:2015
- Position: Front face, easily accessible

**Operation:**

1. **Activation:** Press the red button
   - Immediate stop of all movements
   - Spindle shutdown
   - Machine control software lockout
2. **Reset:** Turn the button clockwise
   - The program must be restarted manually

#### 2.3.2 Limit switches {#232-limit-switches}

**Configuration:**

- 4 mechanical NC limit switches

**Triple function:**

1. **Homing:** Machine referencing at startup
2. **Safety:** Emergency stop if limit reached
3. **Square gantry:** Gantry squaring

- Protection against tool access

## 3. TECHNICAL DESCRIPTION {#3-technical-description}

### 3.1 Overview {#31-overview}

The MEKANIKA EVO is a 3-axis CNC milling machine, allowing material to be machined by moving along the X, Y, and Z axes, or a combination thereof by linear interpolation.

#### 3.1.1 Axis diagram {#311-axis-diagram}

<img src="Media/EVO Schema des axes.png" alt="Axis diagram" />

**Axis terminology:**

- **X-axis:** Lateral movement (left-right)
- **Y-axis:** Longitudinal movement (front-back)
- **Z-axis:** Vertical movement (up-down)

#### 3.1.2 Control box diagram {#312-control-box-diagram}

<img src="Media/EVO Schema du boitier de controle.png" alt="Control box diagram" />

**Connector description:**

- **1**: Fan
- **2**: Motors
- **3**: Limit switches
- **4**: Emergency stop
- **5**: Tool probe (Input 1)
- **6**: Spindle control
- **7**: Input 2
- **8**: Input 3
- **9**: Laser control
- **10**: HDMI
- **11**: USB 2.0
- **12**: Ethernet
- **13**: Power supply
- **14**: Ground

## 6. OPERATION {#6-operation}

The Evo CNC milling machine is intended for occasional and semi-professional use. The machine's design allows for processing a wide variety of different materials, such as wood and non-ferrous metals. However, feed rates and tool engagement will remain modest.

The plausible and reasonable scope of uses includes:

- Manual displacement mode, using a digital "jog" to move the machine along its 3 axes independently.
- Automatic positioning procedure, performed by the user by clicking a digital button. This procedure moves the machine sequentially on each of its axes in the following order: Z, X, Y, until it encounters the limit switches.
- Tool length measurement procedure, activated by the user by clicking a digital button. This procedure lowers the machine at reduced speed along its Z-axis until its tool makes contact with a removable sensor previously placed by the user.
- Normal operating mode, where the machine moves automatically according to machine language, called G-code, previously loaded by the user.
- Manual control mode using a terminal allowing the machine to move according to machine language, called G-code, entered directly by the user.

The plausible and reasonable scope of interventions includes:

- Manual movement of the machine to access certain areas.
- Calibration of the belt and wheel system using a 10mm flat wrench.
- Tool change on the spindle.
- Installation or removal of the dust shoe.
- Installation or removal of material to be milled.
- Starting, pausing, and stopping G-Code.
- Connection/disconnection of external machine cables.
- Opening cable chains to access the machine.

### 6.1 Checklist for safe machine operation {#61-checklist-for-safe-machine-operation}

#### 6.1.1 Daily startup {#611-daily-startup}

**1. Inspection and equipment**

- Inspect the equipment to ensure there are no obvious defects, damaged collets, dull or cracked tools
- Use appropriate PPE (glasses, helmet, ...)
- Tie back long hair and wear well-fitted clothing and roll up your sleeves to avoid snags

**2. Machine preparation**

- Check where the emergency stop button is located
- Never connect or disconnect cables from the electronic control box when the device is powered on
- Ensure that the workpiece is firmly and securely clamped
- Check that the end mill is well sharpened and firmly tightened in the spindle collet
- When handling or changing end mills, avoid touching the cutting edges
- Keep your hands free during manual adjustments

**3. During use**

- Never touch the machine when it is running and keep your hands away from any moving parts during milling
- Stay away from the machine when it is operational

**4. After milling**

- Use a brush or vacuum to clean chips from surfaces
- If V-slot profiles are dusty, blow them out with compressed air to keep them clean and not obstructed by chips
- Clean the floor and work area around the machine
- Clean and store personal protective equipment
- Mark any defective equipment and contact us if you encounter a problem

**5. Normal shutdown**

1. Complete current milling
2. Stop spindle
3. **Shut down Raspberry Pi** (clean shutdown)
4. Wait 30 seconds
5. Turn off control box

> **<i class="fas fa-info-circle" style="color: #2196f3;"></i> CAUTION** - Follow the order to avoid corruption of the Raspberry Pi SD card

**6. Emergency stop**

**In case of danger:**

1. **Press immediately** on the red button
2. Cut power if necessary
3. Analyze the situation
4. Correct the problem
5. Reset (turn the button)
6. Perform a homing procedure again

### 6.3 Tool installation {#63-tool-installation}

#### 6.3.1 Tool change {#631-tool-change}

**Procedure:**

1. Wait for complete stop or machine pause
2. Move Z-axis to high position
3. Unlock ER16 nut (wrenches provided)
4. Remove tool (gloves strongly recommended)
5. Insert new tool
6. Tighten firmly (without excess)

#### 6.3.2 Tool length measurement {#632-tool-length-measurement}

**With the provided tool probe:**

1. Place the probe on the workpiece or spoilboard (depending on the Z0 coordinate defined in the program)
2. Position the spindle above the probe
3. Press the tool length measurement icon
4. Slow automatic descent
5. Contact and memorization of the new Z0 position

### 6.4 Programming and milling {#64-programming-and-milling}

#### 6.4.1 Accepted formats {#641-accepted-formats}

- **G-code:** .nc, .ngc, .gcode
- **Generation:** Via any CAM software. Some examples:
  - Fusion 360 (recommended)
  - VCarve
  - FreeCAD
  - Others

#### 6.4.2 Operating modes {#642-operating-modes}

**Automatic mode:**

1. Load G-code file
2. Check simulation in PlanetCNC
3. Position workpiece origin (X0 Y0 Z0)
4. Start program (Start)

**Manual mode (JOG):**

- Movement by axes
- Variable speed (10-100%)
- Incremental movement possible (examples: 0.01, 0.1, 1, 10 mm)

**MDI mode:**

- Direct G-code command sending
- Sequence testing
- Precise positioning

### 6.5 Monitoring during milling {#65-monitoring-during-milling}

#### 6.5.1 Control points {#651-control-points}

- Abnormal vibrations
- Unusual noise
- Chip quality
- Temperature near tool
- Workpiece clamping

#### 6.5.2 Anomalies and actions {#652-anomalies-and-actions}

| Anomaly          | Probable cause                     | Immediate action                 |
| :--------------- | :--------------------------------- | :------------------------------- |
| **Vibrations**   | Worn tool or improper cut settings | STOP → Check tool and parameters |
| **Smoke**        | Worn tool or improper cut settings | EMERGENCY → Stop                 |
| **Loud noise**   | Very worn or broken tool           | EMERGENCY → Stop                 |
| **Moving piece** | Insufficient clamping              | EMERGENCY → Stop                 |

> In case of doubt → **IMMEDIATE STOP**

### 8.2 Lubrication {#82-lubrication}

A linear system must be sufficiently lubricated to function properly and avoid excessive wear or overheating. The required lubrication interval is determined by environmental conditions.

In general, lubrication is necessary every 100 to 200 hours of operation. We recommend the use of lithium-based greases, which perform well in high-load applications and have good oxidation stability.

**Procedure:**

1. Clean before greasing (cloth and isopropyl alcohol)
2. Apply grease moderately
3. Perform 10 complete machine movement cycles
4. Wipe off excess grease
